Welcome to the Commodore C64DX/C65 System Specifications Manual

This the System Specification manual written by Fred Bowen for the Commodore 65, the successor of the Commodore 64. The machine was never finished, but some pre-production prototype versions of it were sold during Commodore's liquidation in 1993.

I have no longer any sort of idea from where I got the scanned copies of this manual some years ago. If you happen to know :), feel free to remind so I can give the credit where it belongs.
