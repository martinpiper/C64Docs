The BAUD RATE can be generated using the following formulas: 

		    URCLK			      URCLK
    BaudRate = ----------------   or,	 COUNT = ---------------
		16 ( COUNT+1 )			  16 x BaudRate

    Where: COUNT = value loaded into BAUD RATE register
	   URCLK = C7Mhz input, 7.15909 MHz NTSC
				7.09375 MHz PAL

The following tables show some of the most common data rates. Data rate errors of less than +/-1.5% are acceptable for most purposes.
