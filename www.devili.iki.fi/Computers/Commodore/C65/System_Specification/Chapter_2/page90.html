System Specification for C65Fred BowenMarch 1, 1991

Number of cycles per line

In "slow" CPU mode... 

    Total cycles no video                     65
    40 column SCM, MCM, ECM, BMM,
    320 pixel BPM, BPO-BP3 only, or
    640 pixel BPM, BPO-BP1 only     no cost    0
    80 column SCM, MCM, ECM, BMM,
    320 pixel BPM, BPO-BP7, or
    640 pixel BPM, BPO-BP3          substarct 40
    Sprites                         substract  2 per active sprite
