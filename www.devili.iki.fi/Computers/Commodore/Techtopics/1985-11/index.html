ISSUE 11, 1985 : DISK DRIVE 3

MODEL : 1541 "B"

A NEW VERSION OF 1541 DISK DRIVE HAS BEEN RELEASED AND TERMED THE 1541 B. THE 1541 B USES A NEW PCB AND DRIVE ASSEMBLY. A SUPPLEMENT TO THE 1540/1541 SERVICE MANUAL IS ATTACHED. PLEASE INSTALL THESE PAGES IN YOUR MANUAL.
The 1541 B remains operationally the same as the 1541. However, structurally the assemblies used to build the unit are different from the previous models. The assembly parts are NOT INTERCHANGEABLE.

1541 B ASSEMBLY PARTS

PCB ASSY250448-01
NEWTRONICS DRIVE ASSY251643-03
POWER SUPPLY ASSY251868-01

The casework and accessory parts remain the same as the 1541.
